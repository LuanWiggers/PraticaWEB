Escrevendo um parágrafo em HTML

Esta TAG define um parágrafo para um texto escrito em uma página
HTML podendo este conter apenas uma ou mais linhas. Perceba támbem
que utilizamos a Tag "br" antes do parágrafo, para pular linha do trecho
anterior para este e também na produção do testo para pular as linhas.
Há um outro detalhe, veja que utilizamos a tag "strong" para grifar em negrito.
